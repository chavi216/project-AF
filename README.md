שם הפרוייקט:
מערכת לניטור נתוני טיסה


טכנולוגיות:
Backend: Node.js, Express, MongoDB.
Frontend: React, TypeScript, CSS.

הוראות הרצה

את השרת מריצים: נכנסים לתקייה של הבקנד ומריצים node server.js

את הצד לקוח מריצים: נכנסים לתקייה הפורנטד ומריצים npm run dev
במקרה שהספריות לא מותקנות ב2 התקיות גם בפורנט וגם בבקנד צריך להריץ npm install
כמו כן צריך בשרת להגדיר קובץ חדש בשם .ENV שבו נמצא ההגדרות לחיבוא של הדיבי

מבנה הנתונים:
השדות שהמערכת קולטת הם: ADI HSI ALT
כאשר הערך של ALT הוא בין 0 ל3000
הערך של HSI הוא בין 0 ל360
הערך של ADI הוא בין 100 למינוס 100
